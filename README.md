# plan27erp

## Quickstart

1. Install golang - https://go.dev/doc/install
2. Download and install hugo - https://gohugo.io/installation/
3. Start the hugo server locally - `hugo serve`

## Hextra Theme
The theme has been statically injected into the repo.  It is not a submodule/import.  This will help prevent accidental updates to the theme from the upstream source.

### Files Customized:
- `layouts/_shortcodes/card.html`
- `layouts/_partials/shortcodes/card.html`
- `layouts/_partials/navbar.html`
- `layouts/_partials/theme-toggle.html`
- `assets/css/custom.css`
